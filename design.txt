REGISTER
POST /auth/register
BODY fullName, username, emailOrMobile, password, comfirmPassword
RESPONSE
201 { accessToken: String }
400 { message: String }
500 { message: String }

LOGIN
POST /auth/login
BODY emailOrMobile, username, password
RESPONSE
200 {accessToken: String, user: {id: Number, fullName: String, userName: String}}
400 {message: String}
500 {message: String} 

GET AUTHENTICATED user
GET /auth/me
HEADER AUTHORIZATION Bearer ACCESS_TOKEN
RESPONSE
200 {user: {id: Number, fullName: String, username: String}}
401 {message: String}
500 {message: String}

UPDATE PROFILE
PATCH /user
HEADER AUTHORIZATION Bearer ACCESS_TOKEN
BODY profileImage?
RESPONSE
200 { profileImage?: String}
400 { message: String }
401 { message: String }
500 { message: String }
